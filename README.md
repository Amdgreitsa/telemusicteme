# BetterPlayer Telegram

Тема для PulseSync с оформлением полноэкранного плеера в стиле Telegram: тёмная палитра, голубые акценты и более читаемые элементы управления.

## Что меняет тема

- Telegram-палитру для fullscreen-плеера.
- Более контрастные кнопки и состояния `hover/active`.
- Акцентный прогресс-бар и ползунок.
- Улучшенную подложку для блока информации/очереди/текста.

## Настройки темы (что за что отвечает)

Ниже описаны все настройки из `handleEvents.json` и где они используются в `script.js`.

### 1) `playerBackground` — фон из обложки трека
- **Тип:** `button`
- **По умолчанию:** `true`
- **Что делает:**
  - `true` — фоном fullscreen-плеера становится обложка текущего трека.
  - `false` — используется адрес из `backgroundImage`.

### 2) `backgroundImage` — пользовательская ссылка на фон
- **Тип:** `text`
- **По умолчанию:** пусто
- **Что делает:**
  - Позволяет указать свой URL изображения.
  - Работает, когда `playerBackground = false`.

### 3) `autoBlackout` — авто-коррекция яркости фона
- **Тип:** `button`
- **По умолчанию:** `true`
- **Что делает:**
  - Автоматически затемняет слишком светлые обложки, чтобы текст и кнопки оставались читаемыми.

### 4) `brightness` — ручная яркость фона
- **Тип:** `slider`
- **По умолчанию:** `60`
- **Диапазон:** `20..100`
- **Что делает:**
  - Меняет общую яркость фонового изображения в fullscreen-плеере.

### 5) `blur` — размытие фона
- **Тип:** `slider`
- **По умолчанию:** `24`
- **Диапазон:** `0..60`
- **Что делает:**
  - Управляет силой blur для фонового изображения.

### 6) `betterPlayer` — кастомный layout управления
- **Тип:** `button`
- **По умолчанию:** `true`
- **Что делает:**
  - Включает переработанный блок кнопок (очередь, меню, лайк, текст).
  - При выключении возвращает стандартный layout.

### 7) `playerButtonsBackground` — подложка для контента
- **Тип:** `button`
- **По умолчанию:** `true`
- **Что делает:**
  - Добавляет полупрозрачную «стеклянную» подложку и скругления к блокам:
    - текст,
    - информация о треке,
    - очередь.

### 8) `playerButtonsInvertBackground` — инверсия подложки
- **Тип:** `button`
- **По умолчанию:** `false`
- **Что делает:**
  - Дополнительно инвертирует backdrop-слой (эффект для некоторых обложек).
  - Имеет смысл только при включённом `playerButtonsBackground`.

---

## Рекомендации по настройке

- Если фон слишком «шумный»:
  - включите `autoBlackout`,
  - поднимите `blur` до 28–36,
  - снизьте `brightness` до 45–55.
- Если хотите статичный стиль:
  - выключите `playerBackground`,
  - задайте картинку в `backgroundImage`.

## Файлы темы

- `style.css` — визуальная часть Telegram-стиля.
- `script.js` — логика темы и реакция на изменения настроек.
- `handleEvents.json` — набор настроек, которые видит пользователь в PulseSync.
- `metadata.json` — метаданные темы.
